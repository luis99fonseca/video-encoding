\doxysubsection{Detailed Description}
This class implements the \mbox{\hyperlink{classGolomb}{Golomb}} codification to compress information. It encodes an integer based on a given value of M. The codification is the result of the concatenation of an unary code with a binary code, given by the quotient and remainder of the division between integer value and M, respectively.
